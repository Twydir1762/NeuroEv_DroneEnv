# Управление автономным дроном с помощью Q-Learning
Лабораторная работа по исследованию и сравнению трех типов табличных RL-агентов (`RationalAgent`, `ProspectTheoryAgent`, `RiskSensitiveAgent`) в стохастической среде. Для трекинга экспериментов и сохранения артефактов (графиков и отчетов) используется MLflow.

## Инструкция по запуску
### 1. Клонирование репозитория
```bash
git clone https://github.com/Twydir1762/NeuroEv_DroneEnv.git
cd NeuroEv_DroneEnv
```

### 2. Настройка окружения и установка зависимостей
Рекомендуется использовать виртуальное окружение:

```bash
# Создание виртуального окружения
python -m venv .venv

# Активация (для Windows)
.venv\Scripts\activate

# Активация (для Linux/macOS)
source .venv/bin/activate

# Установка библиотек
pip install -r requirements.txt
```

### 3. Запуск локального сервера MLflow
Перед запуском обучения необходимо запустить сервер для логирования метрик:
```bash
mlflow server --port 5000
```

### 4. Запуск обучения и тестирования
В отдельном окне терминала (с активированным виртуальным окружением) запустить основной скрипт обучения:
```bash
python q_learning.py
```
